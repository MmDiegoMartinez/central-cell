📱 Cómo Vincular un Smartwatch con el Teléfono del Cliente


Sigue estos pasos correctamente para que la vinculación sea exitosa y el cliente quede satisfecho.

✅ Pasos Correctos


🔹 PASO 1 — Activar Bluetooth


- Activa el Bluetooth en el teléfono del cliente antes de comenzar.



🔹 PASO 2 — Encender el Smartwatch


- Enciende el smartwatch y espera a que cargue la pantalla principal.



🔹 PASO 3 — Escanear el Código QR


- En el reloj aparecerá un código QR.


👉 Escanear ese código con el teléfono del cliente.



🔹 PASO 4 — Elegir Sistema Operativo


Se abrirá una página donde se debe elegir el sistema:


  • Android
  • iPhone (iOS)


👉 Descargar la aplicación correspondiente.



🔹 PASO 5 — Configurar la App


Abrir la aplicación y:


  • Aceptar todos los permisos
  • Llenar los datos básicos del cliente (nombre, edad, etc.)



🔹 PASO 6 — Vincular Dispositivo


Dentro de la app, entrar a:


👉 "Vincular dispositivo" o "Agregar dispositivo"



🔹 PASO 7 — Seleccionar el Smartwatch


La aplicación buscará equipos cercanos.


👉 Seleccionar el smartwatch correspondiente.


(Si solicita permisos adicionales, se deben aceptar)



🔹 PASO 8 ✅ — Finalizar Vinculación


Esperar a que finalice el proceso de vinculación.


🎉 ¡El smartwatch queda listo para usarse con el teléfono del cliente!



❌ Importante


- Si no se aceptan todos los permisos, la vinculación puede fallar o tener funciones limitadas.
- Asegúrate de que el Bluetooth permanezca activo durante todo el proceso.
